Easy, Fast, Open

Built on Electron, open on Github.

Proton was built out of neccessity. As a college student, I needed a way to quickly take rich notes in class. I have long been a proponent of the Markdown Language, so I decided to download MacDown, which worked wonderfully for my purposes. However, upon a System Update, MacDown stopped rendering Markdown correctly. So, I decided to build my own open-sourced version that is supported cross platform using the Electron Framework.
